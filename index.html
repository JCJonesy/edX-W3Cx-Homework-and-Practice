The Tags!

<!DOCTYPE> is not a tag despite the brackets. This is a DTD. <!DOCTYPE html> should be the first line of code when building a web page as it tells the web browser the following file is an HTML5 file - much easier than the previous, HTML4 DTD: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> - where "strict" would be either strict, transitional, or frameset.

Special Characters

Character Entities are code for characters you wish to type rather than using the keyboard character directly. For example, suppose you want <p> to appear as text on your page. Typing that directly will result in creating a paragraph element rather than displaying it on the screen. For such a special case, use character entities - code that replaces the keyboard character. Therefore, &lt;p&gt; in code would display <p> in a browser.
Something to think about: To display &lt;p&gt; on the screen, the code was: &amp;lt;p&amp;gt; .

There are 5 special characters that should always use entities:
CHARACTER ENTITY NAME ENTITY NUMBER
< &lt; or &LT; &#60; or &#x0003C;
> &gt; or &GT; &#62; or &#xooo3E
& &amp; or &AMP; &#38; or &#x00026;
nbsp; &nbsp; &#160; or &#x000A0;
" &quot; or &QUOT; &#34; or &#x00022;

Of course there are times when using entities is your only option. If the keyboard key combination to produce the desired character is unknown or does not exist, entities are the way to go. To see the possibilities, visit:
